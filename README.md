# 🇹🇷 OTAK (Yerli Yapay Zeka Dil Modeli Ailesi)

OTAK, Türkiye'nin yapay zeka alanındaki dışa bağımlılığını tamamen bitirmek amacıyla sıfırdan kurgulanmış, açık kaynaklı milli bir dil modeli projesidir. 

Temel vizyonumuz, küresel teknoloji devlerinin modellerine olan bağımlılığımızı yenmek ve ülkemizin kendi öz kaynaklarıyla, kısıtlı donanımlarda bile kendi kendine yetebilecek bir yapay zeka altyapısı oluşturmaktır.

---

## 🚀 Otak Mini-1 (Prototip)

Büyük modeller öncesi altyapıyı test etmek amacıyla eğitilen 270 Milyon parametreli ilk sürümümüzdür. Google Colab T4 üzerinde yürütülen ön eğitimde, modelin Loss değeri **3.06** seviyesine indirilmiştir.

*   **Mimari:** Sıfırdan Llama Mimarisi (`LlamaForCausalLM`)
*   **Parametre Boyutu:** 269.89 Milyon (270M)
*   **Eğitilen Veri Boyutu:** 3.2 Milyar Token
*   **Kelime Dağarcığı (Tokenizer):** `ytu-ce-cosmos/turkish-gpt2`
